COMMENT AJOUTER VOS IMAGES
===========================

1. Placez vos images dans ce dossier /images/

2. Nommez-les clairement, par exemple :
   - collection-1.jpg
   - portrait.jpg
   - fondateur.jpg
   
3. Les images actuelles à remplacer :
   - Photo de la collection (about.html ligne ~109)
   - Photo du fondateur (about.html ligne ~114)

4. Formats recommandés : JPG, PNG, WebP
5. Taille recommandée : max 2000px de largeur

Ensuite, modifiez les URLs dans about.html !
